# Euro Gas Network Monitor

Interactive web-based visualization of the European gas infrastructure network.

This project displays major gas pipelines and LNG terminals across Europe, including key attributes such as capacity, operational status, and flow dynamics.

## Features

- Interactive map built with Leaflet.js
- Visualization of:
  - Major gas pipelines
  - LNG terminals
- Dynamic styling based on:
  - Capacity (line thickness / node size)
  - Operational status (active, reduced, closed, sabotaged)
- Tooltips with detailed infrastructure information
- Responsive full-screen layout
- Dark themed cartographic design

## Data Sources

- ENTSOG Transparency Platform  
- Global Energy Monitor (GEM)  
- Eurostat Energy Database  
- Harvard Dataverse (Gas Network datasets)

## Technologies Used

- HTML5
- CSS3 (glassmorphism UI)
- JavaScript (vanilla)
- Leaflet.js (mapping library)
- CartoDB basemaps

## Deployment

This project is designed as a standalone HTML file and can be hosted on:
- GitHub Pages
- Azure Static Web Apps
- Any static web hosting service

## Usage

1. Open the hosted URL
2. Explore the map interactively:
   - Hover over pipelines and terminals
   - Zoom to reveal labels
   - Analyze flow patterns and capacity
